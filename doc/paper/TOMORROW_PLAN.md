### 2.2 设计Few-shot示例
基于实际数据设计3-5个自然语言查询示例：

```
示例1: "查找所有在XX公司工作的员工"
→ MATCH (p:Person)-[:WORKS_AT]->(c:Company {name: $name}) RETURN p

示例2: "查找Alice的同事"
→ MATCH (a:Person {name: 'Alice'})-[:WORKS_AT]->(c)<-[:WORKS_AT]-(colleague) 
  RETURN colleague

示例3: "查找XX部门的所有项目"
→ ...
```

### 2.3 更新Prompt模板

- [ ] 添加针对该数据集的Few-shot示例

- [ ] 简单节点查找（"查找名叫Alice的人"）
