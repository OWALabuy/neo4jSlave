## LaTeX编译环境

确保系统已安装：
```bash
# Ubuntu/Debian
sudo apt-get install texlive-full texlive-xetex

# 或使用TinyTeX
wget -qO- "https://yihui.org/tinytex/install-bin-unix.sh" | sh
```

编译命令：
```bash
cd doc/paper
xelatex main.tex
bibtex main
xelatex main.tex
xelatex main.tex
```
